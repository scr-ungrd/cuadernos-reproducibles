# Cuadernos Reproducibles de Conocimiento del Riesgo de Desastres

Colección de cuadernos computacionales reproducibles sobre gestión del riesgo de desastres en Colombia, desarrollados por la Subdirección para el Conocimiento del Riesgo de la UNGRD.

## Cuadernos disponibles

| # | Título | Amenaza | Región |
|---|--------|---------|--------|
| CRC-001 | Proyección de Temperatura bajo Escenarios de Cambio Climático | Cambio climático | Montería, Córdoba |
| CRC-002 | Riesgo por Olas de Calor Urbano | Calor extremo | Cartagena, Bolívar |
| CRC-003 | Simulación del Tsunami de Tumaco 1979 | Tsunami | Pacífico colombiano |
| CRC-004 | Análisis de Sismicidad de Colombia | Sismo | Nacional |
| CRC-005 | Detección de Inundaciones con Sentinel-1 (GEE) | Inundación | Caribe colombiano |

## Ejecución interactiva

Todos los cuadernos se pueden ejecutar en la nube sin instalación local:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/masantizof/cuadernos-reproducibles-ungrd/HEAD)

## Desarrollo local

```bash
# Instalar MyST CLI
npm install -g mystmd

# Instalar dependencias Python
pip install -r binder/requirements.txt

# Iniciar servidor local
myst start
```
